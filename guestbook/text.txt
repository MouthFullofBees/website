By default Strapi protects APIs.

Go to Settings → Roles → Public

Under Permissions, enable:

find (list all blogs)

findOne (get single blog by slug/id)

Save.

Now you can access blogs at:

http://localhost:1337/api/blogs → list all

http://localhost:1337/api/blogs/1 → single blog
